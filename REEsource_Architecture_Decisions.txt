1. DATA SOURCES & SCOPE
--------------------------------------------------------------------------------
* Primary Goal: Identify potential Rare Earth Element (REE) deposits by locating 
  alkaline intrusions and carbonatites using geophysical proxy data.
* Selected Surveys: 
  - Airborne Magnetic and Radiometric Survey, Colorado Mineral Belt, Mid Block
  - Airborne Magnetic and Radiometric Survey, Colorado Mineral Belt, NE Block
  - Airborne Magnetic and Radiometric Survey, Sierra Madre / Medicine Bow Mountains

2. TARGET FILE TYPES (THE INGESTION PAYLOAD)
--------------------------------------------------------------------------------
* Magnetic Features:
  - Reduced-to-Pole (RTP / RTPCOG): Aligns the magnetic anomaly over the physical source.
  - First Vertical Derivative (1VD / VDCOG): Edge-sharpening high-pass filter.
* Radiometric Features:
  - Equivalent Thorium (eTh / Th): Primary REE proxy.
  - Equivalent Uranium (eU / U): Secondary proxy.
  - Potassium (K): Background geology baseline.
* Excluded Data to Reduce Processing Overhead: 
  - Raw flight-line point data (massive CSVs).
  - Radiometric spectra data (NetCDF / .nc files).
  - Pre-calculated radiometric ratio maps.
